# FinTech_Module_2_Challenge

This project contains a multi-part, modularized app which can be used (along with client data) to identify loans for which the client qualifies. 

---

## Technologies

This program uses Python 3.7, CSV.py, Fire 0.4.0, Questionary 1.9.0, pytest 6.2.4, termcolor 1.1.0 and Path from pathlib. It was written on macOS Catalina 10.15.7.
